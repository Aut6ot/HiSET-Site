History of Space Exploration Technology

Hello, my name is Autumn Gamache. This is a project for my CSE 5953 - Independent Study class. Over this Fall 2025 semester I've been studying the history of modern space exploration, ranging from 1950 to today in 2025. There are undoubtedly space projects that I have missed, because there's so many, but I did my best to cover the ones that I could find.

As for the technical details, I am using GitHub Pages to host the site. I frankensteined my own solution to image hosting for free, but I've never tried it before so I hope it doesn't break things.

This site is best experienced on a computer.
